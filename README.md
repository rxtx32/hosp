 0Here, it is the basic Application for the Capstone Project to publish a Website in the Github for the Hospital Management System.

**Algorithm: Hospital Management System**
1) Start
2) Enter Patient ID, Name, Age, and Disease.
3) Check whether all fields are filled.
4) If any field is empty, display "Please fill all fields" and stop.
5) Store the patient details in the patient list.
6) Display the patient details in the table.
7) Clear the input fields.
8) Repeat the process for new patients.
9) Exit
    
Input: Patient ID, Name, Age, Disease









Output: Patient details displayed in the records table.
